# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.6.x   | :white_check_mark: |
| < 0.6   | :x:                |

## Reporting a Vulnerability
If you believe you've found a security vulnerability in this project, please report it **privately**:

- Send email to: `security@sondera.ai`
- Please **do not open a public GitHub issue** for security vulnerabilities.

### What to include
- A clear description of the vulnerability
- Minimal steps to reproduce
- Impact and risk assessment
- Any relevant logs or evidence

## Response
We aim to respond within 3 business days with next steps.
